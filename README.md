# Meta keys

## Shortcuts
- Ctrl + B toggle table

## Navigation keys:
- 'W' up
- 'S' down
- 'A' left
- 'D' right

## Commands
- 'E' chose color from table
- 'R' change shade of selected table
  
 # Description
 Logic is based on manipulation with crt array which is used to manage video memory nad hex color values
